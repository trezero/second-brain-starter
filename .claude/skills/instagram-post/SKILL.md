# Instagram Post Creator

Create engaging Instagram content for tech/AI education.

## Definition of Done

Post is ready when:
- [ ] Hook stops scroll (first 3 seconds / first slide)
- [ ] One clear topic per post
- [ ] Definition of Done for viewer (what they'll learn/gain)
- [ ] CTA included
- [ ] Hashtags selected (3-5 relevant)
- [ ] Correct format for content type

## Format Selection

| Content | Format | Why |
|---------|--------|-----|
| Tutorial, tips list, guide | **Carousel** (8-10 slides) | 10% engagement rate (highest) |
| Quick tip, trending content | **Reel** (15-30 sec) | Best reach to new audience |
| Updates, engagement | **Stories** | Relationship building |

## Carousel Structure

**Slide 1 - Hook:**
```
[Bold statement or question]

[Subtext: what they'll learn]
```

**Slides 2-8 - Value:**
- ONE point per slide
- Minimal text (20% of slide max)
- Consistent design

**Slide 9 - Summary:**
```
Quick recap:
• [Point 1]
• [Point 2]
• [Point 3]
```

**Slide 10 - CTA:**
```
[Action verb] for more [topic]

Save | Share | Follow
```

## Caption Structure

```
[HOOK - first 125 chars are critical]

[VALUE - the insight/tip]

[CTA - one clear action]

.
.
.
[3-5 hashtags]
```

## Hook Formulas

| Type | Template |
|------|----------|
| Problem | "Stop [mistake] if you want [result]" |
| Curiosity | "The [topic] trick nobody talks about" |
| List | "[Number] [things] that will [outcome]" |
| Insider | "What [experts] don't tell you about [topic]" |
| Transformation | "How I went from [bad] to [good]" |

## Reel Script Structure

```
[0-3 sec] HOOK - "Here's why [topic]..."
[3-15 sec] SETUP - Context/problem
[15-45 sec] VALUE - The content
[45-60 sec] CTA - "Follow for more"
```

## Writing Rules

- **First 125 characters** = your hook (cut off after 2 lines)
- **3-5 hashtags** (quality over quantity)
- **One CTA** per post
- **Captions**: 60% short (<150 chars), 30% medium, 10% long
- **Carousels**: 8-10 slides optimal

## Algorithm Tips

- Post Tuesday-Thursday, 9 AM - 2 PM
- Sends/shares = highest algorithm signal
- Saves = strong signal for educational content
- First 3 seconds determine Reel success
- Respond to comments within first hour

## Hashtag Strategy

Use 3-5 from these categories:
- **Industry**: #ArtificialIntelligence #TechEducation
- **Niche**: #AITools #PromptEngineering
- **Content type**: #Tutorial #TechTips

## What NOT to Do

- Multiple CTAs (confuses action)
- Hashtag stuffing (>5 looks spammy)
- Walls of text on slides
- Burying CTA in long caption
- Watermarks from other platforms
